# Engineering Playground
## Introduction
The goal of this project is to learn how a real-life project looks like in order to be able to organise, write and structure proper projects in the future.

## Completion Checklist
To consider this project complete, it must satisfy the following requirements:
- [ ] Project structure created.
- [ ] Git repository configured.
- [ ] Virtual environment configured.
- [ ] README completed.
- [ ] `.gitignore` configured.
- [ ] First Python application working.
- [ ] Project runs from the terminal.
- [ ] Multiple meaningful commits created.
- [ ] Project pushed to GitHub.
- [ ] Project fully documented.
