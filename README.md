# Engineering Playground
## Introduction
The goal of this project is to learn how a professional software project looks like in order to be able to organise, write and structure proper projects in the future.

## Project Functionality
This project will have just a simple functionality. It is just a simple task manager.
**Features:**
- Categories that can be created to organise the tasks.
- Time can be added so it is more organised.
- Completed and uncompleted tasks.
